# servidor_forca
## Requisito **ter o npm instalado**.  
  
## Para rodar a versão V1.0:
* Clonar a tag v1.0 do projeto.
* npm install
* npm run server  
  
## Paa rodar a versão V2.0:  
* Clonar a tag v2.0 do projeto.
* npm install  
* npm run websockets  
  
## Para rodar a versão v4.0:
* Clonar a tag v4.0 do projeto.
* npm install  
* npm run mq
